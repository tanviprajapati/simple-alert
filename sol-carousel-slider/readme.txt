Sol Carousel slider is a WordPress plugin that allows you to create and modify your slider and it's content from admin.


== Description ==

SOl Carousel Slide is a very simple and easy slider. You can just add new side as per your requirement.

You can show your new slider design with any page via below short-code.

**Shortcode** : [wp_sol_carousel_slider] - To display slider on page

> **SOl Carousel Slide Plugin Features**
>
> * Fully Responsive
> * Cross Browsers Support ( Firefox, Chrome, Safari, Opera, etc. )
> * Easily manage slider by simply add slide's title , image and content.
> * You can easily add or delete old slide.



== Installation ==

1. Upload the **ssol-carousel-slider.zip** file Via WordPress Admin > Plugins > Add New,
2. Alternately, upload **sol-carousel-slider** folder to the /wp-content/plugins/ directory via FTP,
3. Activate the **Sol Carousel Slider** plugin from Admin > Plugins.
